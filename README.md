🚀 ORION-X
A WebGL-Based Interplanetary Mission Simulation

🌌 Overview

ORION-X is a cinematic 3D interplanetary mission simulation built using Three.js and WebGL.

The project visualizes a spacecraft traveling from Earth to Mars along a curved transfer trajectory with real-time mission phase tracking and a dramatic Mars orbit insertion sequence — all running directly in the browser.

No backend. No frameworks. Pure web technology.

✨ Features

🌍 3D Rotating Earth

🔴 3D Mars Model

🌠 Deep Space Starfield

🚀 Animated Rocket Transfer Trajectory

📡 Live Mission Phase Updates

📏 Distance Tracking System

🎬 Cinematic Mission Completion Screen

💻 Fully Browser-Based (No Installation Required)

🛰 Mission Phases

Leaving Earth Orbit

Deep Space Cruise

Mars Approach

Mars Orbit Insertion

Mission Successful Sequence

🧠 Technologies Used

HTML5

CSS3

JavaScript

Three.js (WebGL 3D Engine)

🚀 How to Run Locally

Download or clone the repository

Open index.html in your browser

OR

Use VS Code Live Server extension.

🎯 Purpose of the Project

This project demonstrates how advanced space simulations and cinematic visualizations can be built entirely using frontend web technologies.

It is designed for:

Educational demonstrations

Portfolio projects

Science exhibitions

Creative web graphics experiments
